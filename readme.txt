== Description ==

You can display share-buttons at all of single posts and single pages very easily.

== Installation ==

1. Upload the entire `jtpp-share-buttons` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

Then you will find 'JTPP Share Buttons' menu in your WordPress admin panel.
